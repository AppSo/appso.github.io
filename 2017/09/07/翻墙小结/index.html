翻墙方案

Shadowsocks

@clowwindy

Shadowsocks是一种基于Socks5代理方式的网络数据加密传输包。

客户端

  • Windows: Shadowsocks-win / Shadowsocks-Qt5
  • OS X: ShadowsocksX-NG
  • Linux: Shadowsocks-Qt5
  • Android: Shadowsocks-android
  • iOS: Wingy / Surge

ShadowsocksR

@breakwa11

ShadowsocksR是breakwa11发起的shadowsocks分支，在shadowsocks的基础上增加了一些数据混淆方式，修复了部分安全问题并可以提高QoS优先级。

客户端

客户端
